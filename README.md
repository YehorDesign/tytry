# ТИТРИ / TYTRY

Локальний інструмент для субтитрів як на veed.io: завантажуєш пачку відео → Deepgram розпізнає мову з таймінгами по словах → обираєш стиль / правиш текст і тайминги → Remotion рендерить відео з вшитими субтитрами.

## Для дизайнерів: установка

1. Запустити `TYTRY-Setup-1.0.0.exe` (Windows). SmartScreen може попередити про невідомого видавця — «More info» → «Run anyway» (підпису коду немає, це нормально для внутрішнього інструменту).
2. Додаток сам створить ярлик і відкриється.
3. Натиснути ⚙ зверху і вставити Deepgram API key (безкоштовно на console.deepgram.com).
4. Перший рендер довший: докачується Chrome Headless (~110 МБ, один раз).

macOS: збірка `TYTRY-x.x.x.dmg` робиться через GitHub Actions (див. нижче) — на Windows dmg зібрати неможливо.

## Розробка

```bash
npm install
npm run dev        # дев-сервер на localhost:3000
npm run app        # десктоп-вікно з продакшен-сборки (спершу npm run build)
npm run dist       # Windows-інсталятор у dist/
npm run dist:mac   # macOS dmg (тільки на macOS)
```

`ТИТРИ.bat` — швидкий запуск десктоп-версії без інсталятора (збирає за потреби).

### macOS-збірка через GitHub Actions

`.github/workflows/build.yml` збирає exe + dmg на раннерах GitHub. Запуш репозиторій на GitHub → вкладка Actions → «Build installers» → Run workflow. Артефакти з'являться в результатах ранy.

## Можливості

- Пачка відео, авто-транскрибація (Deepgram nova-2: авто / українська / англійська)
- 20 пресетів: класика, Hormozi-підсвітка, плашки, караоке, Beast-мультиколор, неон, маркер, градієнт, комікс, ретро-3D, друкарська машинка, Opus, мінімал + 5 «журнальних» design-пресетів (Cover, Zine, Editorial, Sketch, Poster) — слова різного розміру й шрифту, як обкладинка
- Налаштування: шрифт (вбудовані + всі системні), розмір, позиція, кольори, слів на екрані, великі літери
- Редактор тексту (клік по слову) + таймлайн: тягни блок субтитрів, розтягуй за краї
- Папка збереження на вибір (⚙), черга рендеру, видалення проєктів
- Інтерфейс: українська / English

## Технічне

- Next.js 15 + Remotion 4 (прев'ю і фінальний рендер — один React-компонент)
- Дані: `workspace/` локально (у встановленій версії — `%APPDATA%/TYTRY/workspace`)
- Remotion безкоштовний для компаній до 3 людей; більше — потрібна ліцензія (remotion.dev/license)
